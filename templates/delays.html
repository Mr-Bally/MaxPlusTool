Delays

This is the page where you define the delays for each part of the networks.

This can be altered without modifying the underlying network to mimic the changing traffic depending on the delays on a given day.

When entering the delays please make sure that:

  • The matrix size matches the size of the underlying network matrix
  • If EPS is displayed in the network matrix simply put 0 in its place in the matrix
  • You select if you are entering constants or rates of exponential
  • Click save when you have entered the matrix
