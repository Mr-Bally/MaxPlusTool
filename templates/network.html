Network

This page allows you to create or alter the matrix which represents the network to be tested.

It also allows you to name the stations (making the labels more meaningful on the resulting graph).

When entering the matrix please ensure that:

  • The matrix is square (IE equal number of rows and columns)
  • The matrix is entered in the given format
  • The values are either integer values or EPS
  • If there is no connection between the stations please represent this with EPS (epsilon)


Once you have entered the matrix please click enter matrix to update the station numbers and names

When you have finished the naming the stations then click save to persist the data to the server
